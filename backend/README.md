# ANANDAM Life – Backend (Express + TypeScript)

This folder is a **separate backend scaffold** for the Next.js frontend.

## Run locally

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

Backend will start at `http://localhost:4000` (configurable via `PORT`).

## Endpoints

- `GET /` basic status
- `GET /health` health check
